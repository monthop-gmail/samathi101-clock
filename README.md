# Samathi101 Clock 🧘‍♂️

นาฬิกาจับเวลาสำหรับเดินจงกรม และนั่งสมาธิ รองรับการใช้งานแบบ Offline (Progressive Web App - PWA)

## ฟีเจอร์หลัก (Features)
- 📴 **Offline Support:** ใช้งานได้แม้ไม่มีอินเทอร์เน็ตผ่านระบบ PWA
- ⏱️ **Timer Modes:** มีโหมดจับเวลาวิทิสาสมาธิ, เดินจงกรม, นั่งสมาธิ และปรับเวลาเองได้
- 🔊 **Audio Guides:** เสียงระฆังเตือน, เสียงนำนั่งสมาธิ, และเสียงนำเดินจงกรม
- 🌙 **Dark Mode:** รองรับหน้าจอมืดเพื่อถนอมสายตา

## การติดตั้ง PWA
แอปพลิเคชันจะทำการ Cache ไฟล์หน้า UI และไฟล์เสียงขนาดเล็กไว้ที่เครื่องอัตโนมัติเมื่อครั้งแรกที่เปิด ทำให้สามารถ **ใช้งานบนระบบแบบ Offline ได้ 100%** เพียงแค่กด "Add to Home Screen" ผ่านเบราว์เซอร์
