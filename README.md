# Samathi101 Clock 🧘‍♂️

แอปพลิเคชันเว็บ (PWA) จับเวลาสำหรับเดินจงกรม และนั่งสมาธิ ออกแบบมาเพื่อให้สามารถใช้งานแบบออฟไลน์ได้ 100% พร้อมเสียงนำนั่งสมาธิและสิ่งอำนวยความสะดวกครบครัน

🌐 **Live Demo:** [samathi101-clock.pages.dev](https://samathi101-clock.pages.dev/)

---

## 🎯 ฟีเจอร์หลัก (Features)
1. **ระบบจับเวลาสมาธิ (Meditation Timer)**
   - โหมดวิทิสาสมาธิ, เดินจงกรม, นั่งสมาธิ และจับเวลาทั่วไป
   - สามารถปรับตั้งเวลาเองได้ตามต้องการ (Custom Timer)

2. **ระบบเสียงช่วยเหลือ (Audio Guides & Alerts)**
   - เสียงระฆังเตือนเมื่อเริ่มและจบการทำสมาธิ
   - เสียงนำเดินจงกรม และเสียงนำนั่งสมาธิ (เปิดใช้งานเป็นค่าเริ่มต้น โดยจะเล่นต่อเนื่องหลังจากจบเสียงระฆังโดยอัตโนมัติ)
   - ระบบเตือนหลับ (Sleep Alert) ด้วยเสียงเตือนทุกๆ 5 นาที (สามารถเปิด-ปิดได้)

3. **เทคโนโลยี PWA (Progressive Web App) & Offline Support**
   - สามารถกด "Add to Home Screen" เพื่อติดตั้งลงบนหน้าจอมือถือ/แท็บเล็ตได้ราวกับแอปพลิเคชันพื้นเมือง (Native App)
   - **Service Worker:** ทำการ Cache ไฟล์หน้า UI และ **ไฟล์เสียงทั้งหมด** อัตโนมัติเมื่อครั้งแรกที่เปิด ทำให้สามารถเปิดใช้งานแอปแบบออฟไลน์ได้เวลาที่ขาดอินเทอร์เน็ตสมบูรณ์แบบ

4. **การออกแบบและ UI (Adaptive UI)**
   - โหมดหน้าปัดดิจิทัลและหน้าปัดนาฬิกาเข็มคลาสสิก
   - **Dark Mode:** รองรับโหมดหน้าจอมืดเพื่อถนอมสายตาในที่แสงน้อย

## 📲 วิธีใช้งานแบบออฟไลน์ (Offline Usage)

แอปนี้รองรับการใช้งานแบบ **ไม่ต้องต่อเน็ต** หลังจากเปิดครั้งแรกเพียงครั้งเดียว

### มือถือ Android (Chrome)
1. เปิดเว็บไซต์ในเบราว์เซอร์ **ครั้งแรก** (ต้องมีเน็ต)
2. จะมีแถบขึ้นมาว่า **"เพิ่มไปยังหน้าจอหลัก"** → กดเพิ่ม
3. ถ้าไม่ขึ้น → กดเมนู ⋮ → **"ติดตั้งแอป"** หรือ **"เพิ่มไปยังหน้าจอหลัก"**
4. จะได้ icon บนหน้าจอเหมือนแอปปกติ
5. **ครั้งต่อไปเปิดได้โดยไม่ต้องมีเน็ต** ✅

### มือถือ iOS (Safari)
1. เปิดเว็บไซต์ใน **Safari** (ครั้งแรกต้องมีเน็ต)
2. กดปุ่ม **แชร์** (กล่องมีลูกศรขึ้น) ด้านล่าง
3. เลื่อนหา **"เพิ่มไปยังหน้าจอหลัก"** → กดเพิ่ม
4. จะได้ icon บนหน้าจอ
5. **ใช้ offline ได้เลย** ✅

### คอมพิวเตอร์ (Chrome / Edge)
1. เปิดเว็บไซต์
2. จะเห็นไอคอน **ติดตั้ง** (⊕) ที่ address bar → กด **"ติดตั้ง"**
3. แอปจะเปิดเป็นหน้าต่างแยก ใช้ offline ได้

> 💡 **หลักการทำงาน:** เว็บนี้ใช้ Service Worker ดาวน์โหลดไฟล์ทั้งหมด (HTML, CSS, JS, เสียง) เก็บไว้ในเครื่องตั้งแต่ครั้งแรกที่เปิด → ครั้งต่อไปไม่ต้องโหลดจากเน็ตอีก

## 🚀 การรันโปรเจกต์ (Local Development)
การรันโปรเจกต์นี้บนเครื่องคอมพิวเตอร์แบบออฟไลน์ สามารถทำได้ง่ายๆ เพียงแค่มี Local HTTP Server เช่น:

**หากมี Python:**
```bash
python -m http.server 8000
```
แล้วเปิดเบราว์เซอร์ไปที่ `http://localhost:8000`

**หากใช้ VS Code:**
แนะนำให้ติดตั้ง Extension อย่าง **Live Server** และทำเพียงคลิกเปิดจากไฟล์ `index.html` ก็จะทำงานได้ทันที

---

## ⚙️ ระบบ CI/CD (Continuous Deployment)
โปรเจกต์นี้ผูกระบบอัตโนมัติเข้ากับ **GitHub Actions** (ไฟล์ `.github/workflows/deploy.yml`)
ทุกครั้งที่มีการอัปเดตโค้ดลงบน branch `master` ระบบจะทำการ Deploy โค้ดใหม่ขึ้นสู่บัญชี **Cloudflare Pages** โดยทันที

**สิ่งสำคัญเมื่อนำไปใช้เอง (Environment Secrets):**
ต้องมีการตั้งค่าใน Repository Secrets ของ GitHub ดังนี้:
- `CF_API_TOKEN`: Cloudflare Global/API Token ที่มีสิทธิ์สั่งอัปเดตระบบ Pages
- `CF_ACCOUNT_ID`: Cloudflare Account ID

## 🔄 การตั้งค่า Redirect จากโดเมนเก่า
หากคุณต้องการย้ายผู้ใช้จากเว็บไซต์เดิม (`https://sumana.org/clock/`) มายังแอปใหม่นี้บน Cloudflare Pages แบบเนียนๆ (ผู้ใช้เข้าเว็บเก่าแล้วเด้งมาเว็บใหม่ทันที) และเป็นผลดีต่อ SEO แนะนำให้ใช้วิธีแก้ไขไฟล์ `.htaccess` บนเซิร์ฟเวอร์เก่า:

1. เข้าไปที่โฟลเดอร์ของเว็บไซต์เก่า (เช่นโฟลเดอร์ `clock`)
2. สร้างหรือแก้ไขไฟล์ชื่อ `.htaccess` (อย่าลืมจุดนำหน้า)
3. ใส่โค้ดนี้ลงไป:

```apache
Redirect 301 /clock https://samathi101-clock.pages.dev/
```
*(หากใส่ในโฟลเดอร์ `clock` โดยตรง อาจใช้แค่ `Redirect 301 / https://...` ขึ้นอยู่กับการตั้งค่าโฮสติ้ง)*
