# Git & GitHub Workflow

Always use `wsl` to run git and GitHub CLI (`gh`) commands.

// turbo-all

1. Use `wsl` prefix for all git commands, e.g. `wsl git status`
2. Use `wsl gh` for GitHub CLI commands, e.g. `wsl gh repo view`
3. Working directory should be the project root
